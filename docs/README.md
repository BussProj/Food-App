# Project Documentation Index

This directory serves as the root repository for all platform technical specifications, architecture decisions, and operational guides.

## Directory Structure
*   `architecture/`: Holds finalized system blueprints, database architectures, and core real-time designs.
*   `development/`: Contains guides for developer onboarding, environment setups, and coding standards.
*   `api/`: Stores API contract documentation, JSON payload samples, and WebSocket event structures.
*   `deployment/`: Contains SRE playbooks, container configurations details, and disaster recovery specs.
*   `decisions/`: Stores Architectural Decisions Records (ADR) detailing key design shifts and rationale.
