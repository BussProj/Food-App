# System Architecture Documentation

This directory contains the frozen, finalized architectural blueprints and technical specifications for the platform. These documents represent the permanent system guidelines and must be followed strictly during implementation.

## Finalized Architecture Blueprints
1.  **Product Understanding**: Outlines the product vision, priority pillars, and business model.
2.  **Enterprise System Architecture**: Defines modular boundaries, Ports & Adapters, and dynamic context injections.
3.  **Database Architecture**: Outlines the polyglot persistence schema, tenant isolation, and lifecycle archiving.
4.  **Backend Technical Blueprint**: Details the Django project structure, request/response middleware, and worker queues.
5.  **Frontend Technical Blueprint**: Details the Angular PWA, Signal state services, and dynamic styles variables.
6.  **API Technical Blueprint**: Sets JSON format protocols, HTTP headers, and WebSocket sync mappings.
7.  **Real-Time Technical Blueprint**: Outlines Channels, Redis namespaces, sequence checks, and offline catch-ups.
8.  **Security Technical Blueprint**: Details CBAC, database RLS locks, override keys, and envelope cryptography.
9.  **DevOps & Deployment Technical Blueprint**: Defines container configurations, rolling deploys, and SRE monitoring.
