# Setup & Automation Scripts

This directory houses development and production automation scripts.

## Directory Structure
*   `dev/`: Contains local dev environment spin-up scripts (e.g. Docker Compose orchestrator wrappers).
*   `setup/`: Holds fresh repository installation utilities and dependencies check modules.
*   `database/`: Houses database schema seed controllers, automated backup schedules, and recovery triggers.
*   `deployment/`: Contains pipeline deployment wrappers and rollback controllers.
