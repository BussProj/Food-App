# Project Assets Directory

This directory stores visual assets, graphic designs, icons, mockups, and logos for the Restaurant Management SaaS Platform.

## Directory Structure
*   `logos/`: Contains brand logos, platform icons, and billing system graphic signs.
*   `icons/`: Stores SVG UI icons, navigation buttons, and touch indicators.
*   `images/`: Stores customer catalog stock photos and system banners.
*   `mockups/`: Houses the visual interface mockups, wireframes, and original designs (from Figma or image exports).
